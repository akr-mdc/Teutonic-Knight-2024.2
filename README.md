# Teutonic-Knight-2024.2
Implementação de Máquinas de Estados Finitos no ambiente Unity.
# Controles
Andar = A/D ou LeftArrow/RightArrow

Correr = Shift

Saltar = W ou UpArrow

Deitar = S ou DownArrow

Golpe básico = J

Golpe secundário = K

Golpe terciário = L

Chute = H

Atirar flecha = I

Incocar bola de fogo = O

Dançar = F
